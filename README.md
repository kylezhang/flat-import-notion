## 项目名
第三方笔记导入白板插件
## 项目简介
使用Notion API从Notion 导入文字内容到白板
## 安装部署指南
npm install
## 功能简介
从Notion 导入文字内容到白板
## 使用
到Notion Developer网页申请机器人
https://www.notion.so/my-integrations 拿到secrets，在想要导入的页面邀请机器人，用网页打开Notion页面，最后一串字串即为pageid。
点击插件按钮，第一个框填secrets，第二个框填pageid，点击导入按钮，即可导入。
## 技术栈
Notion API.
white-web-sdk
